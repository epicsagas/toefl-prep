# TOEFL Writing — Grading Rubric (ETS-aligned, 0–5)

The LLM grades written essays directly from text. This is the strongest local
evaluation fit (text in → text rubric out). Covers both task types.

## Task family

- **Integrated** (Task 1): read passage (~3 min) + listen to lecture (~2 min,
  counters the reading) → write 150–225 word summary relating the two (20 min).
- **Academic Discussion** (Task 2, post-2023): read a prof + 2 students'
  posts → write your own contribution (~100 words, 10 min). Replaces the old
  Independent essay.

## Scoring scale (0–5 per task, ETS)

| Score | Descriptor (condensed) |
|-------|------------------------|
| 5     | Fully addresses; well-organized; detailed, accurate (integrated: relates reading↔lecture precisely); varied, accurate language; few errors. |
| 4     | Well-handled with minor lapses; generally coherent; generally accurate language. |
| 3     | Adequate but uneven; some vagueness or incomplete development; noticeable but non-blocking errors. |
| 2     | Limited development; weak organization; frequent errors that obscure meaning; integrated: weak connection between sources. |
| 1     | Serious weaknesses; little relevant content; severe language errors. |
| 0     | Off-topic, copied, blank, or non-English. |

## Dimensions

1. **Content / Topic Development** (0–5): addresses the prompt fully;
   ideas developed with reasons/examples. Integrated: accurate capture of the
   lecture's rebuttals and correct mapping to reading points.
2. **Organization / Coherence** (0–5): thesis, logical paragraph flow,
   transitions, clear intro/conclusion. Integrated: balanced treatment of
   both sources, not a reading-only summary.
3. **Language Use** (0–5): grammar accuracy, sentence variety, lexical range,
   appropriate register. Penalize systematic errors (subject-verb, articles,
   tense), reward complex structures used correctly.
4. **Mechanics** (0–5, minor weight): spelling, punctuation, capitalization.

## What the evaluator MUST produce

- Per-dimension score with one quoted example from the essay.
- **Synthesis score** (weighted: Content 0.35 / Org 0.30 / Language 0.30 /
  Mechanics 0.05 → mapped to 0–5, one decimal).
- Word count + note if outside the expected band (integrated <150 or
  academic-discussion <75 flagged as under-length).
- **Top 3 line-level fixes** (quote the original line → show a rewrite).
- Integrated: a **source-mapping table** (reading point ↔ lecture counter) so
  the learner sees what they captured vs. missed.

## Honesty rule

Do not reward length or template density alone. If the essay is a memorized
template padded around thin content, cap Content ≤ 2 and say so. Distinguish
genuine development from filler.
